# Replacement-name research

**Search date:** 2026-07-18  
**Product:** Local Chrome extension that extracts the useful text from a webpage and exports clean Markdown  
**Scope:** Broad practical preliminary conflict screening, not a legal clearance opinion

## Recommendation

Use **PagePith**.

**Positioning:** The useful core of a page, in Markdown.  
**Primary tagline:** `Clean pages. Plain Markdown.`  
**Secondary line:** `Keep the useful core.`

`Pith` means the essential or central part of something. The composite is suggestive rather than merely descriptive, it explains the product once seen with the tagline, and the screening below found no material exact or close software conflict.

## Important spelling distinction

The submitted candidate was **Page Morrow**, and that spelling was researched as written. It was not silently changed to the earlier **Page Marrow** proposal.

- **Morrow** means the following day or future. It is distinctive but does not explain extraction, cleaning, or Markdown.
- **Marrow** means the inner substance or core. It fits the product metaphor much better, but some users will first think of anatomy.

Both are included in the comparison.

## 4. Page Marked

### Material findings

- USPTO `PageMarked`: no exact result.
- Exact GitHub repository: `chs97/PageMarked`, created in 2018, for marking a webpage's scroll position.  
  https://github.com/chs97/PageMarked
- Chrome Web Store search returns `PageMark Navigator` and numerous page-marking/highlighting extensions.  
  https://chromewebstore.google.com/search/Page%20Marked
- `pagemarked.com` is registered.
- General web results are dominated by generic page markers, bookmarks, and stationery.

### Assessment

**Do not choose.** The field is crowded, the wording is ordinary, and consumers may assume bookmarking or annotation rather than Markdown extraction. It would be difficult to search for and difficult to protect.

## 5. HTML MD

### Decisive conflict

An exact near-identical Chrome extension already exists:

- **HTML.MD**
- Chrome Web Store: https://chromewebstore.google.com/detail/htmlmd/dbdnbdajoeadlkjghcfbinciabccjifa
- Description: converts any webpage to Markdown, strips scripts/styles, preserves essential text and structure, copies output, and targets AI assistants.
- Observed Store metrics: 36 users and two ratings.

This is essentially the proposed product under the proposed name.

Additional crowding:

- `HTML to MD`, `HTML to Markdown Converter`, `Save HTML as Markdown`, and other direct competitors appear in the same Chrome Web Store search.
- GitHub contains multiple `htmlmd` and `html-md` converter repositories.
- npm includes `html-md-converter` and related packages.
- USPTO `HTMLMD`: no exact result, but that does not cure direct marketplace use or make the descriptive phrase ownable.

### Assessment

**Material conflict; do not choose.** It is both directly occupied and highly descriptive. Even without the exact Store listing, it would be a poor brand because nobody could reliably search for or own it.

## 6. Other alternatives screened

### WebWinnow

No exact `WebWinnow` match was found in the USPTO, Chrome Web Store, GitHub, npm, PyPI, Firefox Add-ons, or Apple App Store checks. However, the USPTO contains several live `WINNOW` registrations in Classes 9 and 42 for software and SaaS, including Winnow Solutions entities and Aya Healthcare. The root is too crowded to justify choosing it over a cleaner coined composite.

### PithMark

No exact federal mark, Chrome extension, GitHub repository, npm package, PyPI project, company, or exact mobile app was found. However, Google Play returns an active **Pitchmark** app/product, one letter away, and search systems may autocorrect `Pithmark` to `Pitchmark`. PagePith avoids that problem.

### PagePlain

No exact software/package collision was found in the initial screen, but `pageplain.com` is registered and serves a lander/parked page. More importantly, `page` plus `plain` is relatively descriptive and weaker as an ownable brand than PagePith.

### Plainleaf

No exact software conflict appeared in the initial screen. It is pleasant but sounds like stationery, tea, or a sustainability product and does not anchor itself to webpages or Markdown.
