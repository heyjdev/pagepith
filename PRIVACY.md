# Privacy policy for PagePith

Last updated: July 18, 2026

PagePith converts the useful text on the webpage you choose into Markdown. The extension performs this work locally in your browser.

## Data collection

PagePith does not collect, transmit, sell, or retain:

- webpage content
- browsing history or URLs
- generated Markdown
- personal information
- analytics or usage data
- device or account identifiers

PagePith has no developer-operated servers, user accounts, telemetry, advertising, or third-party tracking.

## Browser permissions

PagePith uses Chrome's `activeTab` permission to access the current webpage only after you open the extension. It uses the `scripting` permission to run the packaged extraction code in that tab. The extension processes a cloned copy of the page content and does not modify the original webpage.

## Local files and clipboard

When you choose Copy Markdown, Chrome places the generated text on your clipboard. When you choose Export `.md`, Chrome saves a local file through the browser's normal download behavior. PagePith does not receive or retain either result.

## Third parties

PagePith does not send data to the developer or any third party. Its open-source extraction libraries are packaged inside the extension and do not make network requests.

## Changes

If a future version changes these practices, this policy and the Chrome Web Store disclosures will be updated before that version is released.

## Contact

For support, bug reports, or privacy questions, open an issue at [github.com/heyjdev/pagepith/issues](https://github.com/heyjdev/pagepith/issues).
